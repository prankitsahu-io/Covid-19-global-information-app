# Covid19

A Covid 19 globle information Live Data using Flutter.

## Download App

[Android](apk/covid19.apk)

## Api Used
- [Covid19api](https://api.covid19api.com)
- [publicIp](https://api.ipify.org)
- [GeoLocationFromIp](https://ipapi.co)
